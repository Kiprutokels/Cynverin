# Cynverin
Cynverin is your all-in-one personal digital hub that consolidates work, life, and everything in between into a secure, AI-powered platform. Manage events, tasks, projects, finances, documents, and more—all in one place, with intelligent AI features that give personalized insights and smart reminders.

Key Features

Events & Calendar – Schedule, track, and get reminders for personal & professional events.

Tasks & Projects – Organize, prioritize, and monitor your projects and goals.

Document & File Management – Securely store, categorize, and retrieve files with AI-assisted search.

Financial Tracking – Manage budgets, track expenses, and maintain financial records.

AI Assistant – Answer questions, summarize notes, and perform semantic searches across your data.

Embedding & Memory System – AI remembers context to provide personalized insights.

Secure Data Management – Full ownership of your data stored in your PostgreSQL database.

Tech Stack

Frontend: React / Next.js, TailwindCSS

Backend: Node.js + Express

Database: PostgreSQL + Prisma ORM + pgvector

AI: OpenAI API for semantic search & personal assistant features

Deployment: Vercel / Custom server

Why Cynverin?

Cynverin helps you manage your digital life efficiently by combining productivity tools with AI-powered intelligence—making it your truly personal, secure, and responsive digital assistant.
